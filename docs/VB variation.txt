[mesh buffer format]

원본 직접 참조
- burns
- instanceID -> transform, vertex/index buffer base
- triangleIID -> index0, 1, 2 -> vertex attributes
- 별도 버퍼 없음, visibility pass 쌈, static geometry에서 구조가 단순
- 하지만 새 vertex를 gather해야 함, tessellation이나 procedural geometry가 원본 triangle ID와 대응되지 않을수있음, mesh별로 vertex layout 다를수도있음
- 실제로는 SoA방식 global buffer로 통합하거나 meshlet/cluster metadata가 각각 stream offset이나 format decoder를 제공함
